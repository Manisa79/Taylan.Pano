# Gridly v51.1 - Product / Developer Menu Separation

Bu sürümde Gridly örnek menüsü sadeleştirildi.

## Neden?
Gridly artık çok sayıda görünüm, medya, tema, performans ve geliştirici örneği içeriyor. Gerçek uygulamalarda bu örneklerin tamamının ana menüde görünmesi kullanıcıyı gereksiz şekilde yoruyordu.

## Yeni Menü Davranışı
Ana menü artık kısa tutulur:

- Example Center / Arama
- Audix Medya Pilot
- Theme Audit / Okunurluk
- Media Playback State
- Showcase
- Developer Center / Tüm Teknik Örnekler

## Developer Center
Tüm teknik örnekler artık alt gruplara taşındı:

- 5.x Stabilite / Kullanım
- Media / Audix
- Core Grid
- Designer / API / Uyumluluk
- Enterprise / Performance

## Uygulama Projeleri İçin Öneri
Audix, AOI Support Desk, Line Workspace, MasterData ve Bilge Defter gibi gerçek uygulamalarda Gridly örnek menüsü gösterilmemeli. Sadece uygulamanın kendi menüleri görünmeli. Gridly örnekleri ayrı Example Center/Test App içinde tutulmalı.

## Amaç
- Kullanıcı menüsünü kalabalıktan kurtarmak
- Gridly teknik örneklerini kaybetmeden düzenlemek
- Showcase ile hızlı deneme alanı bırakmak
- Geliştirici örneklerini tek yerde toplamak
